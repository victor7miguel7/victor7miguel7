# Sobre mim:
Sou estudante de Ciência da Computação (10º período) na UFRPE e atuo como Analista de Dados, com experiência prática em Python, R, SQL e Power BI. No meu ambiente de trabalho, participo de projetos reais e estratégicos, desenvolvendo dashboards interativos e painéis gerenciais que apoiam a tomada de decisão baseada em dados em áreas como operações, finanças, marketing e gestão de pessoas.

Minha atuação abrange as três principais vertentes da área de dados — Análise de Dados, Engenharia de Dados e Ciência de Dados — permitindo uma visão completa do ciclo analítico, desde a coleta e tratamento das informações até a geração de insights e previsões.

Como Analista de Dados, sou responsável pela extração e análise de informações via SQL, atendendo às demandas de diferentes áreas do negócio, e pela criação de dashboards e indicadores no Power BI, utilizando DAX e linguagem M para transformar dados em insights estratégicos.

Como Engenheiro de Dados, lidero o pipeline de dados corporativo em ambiente AWS, conduzindo todo o fluxo de extração, transformação e carga (ETL) dos dados para um Data Warehouse estruturado para o BI. Desenvolvo scripts em Python para automação de processos, monitoramento de execuções, tratamento de falhas e integração com APIs e bancos de dados, além de ser responsável por gerenciar e otimizar o uso de recursos na nuvem, garantindo eficiência e controle de custos.

Como Cientista de Dados, participo do desenvolvimento e validação de modelos preditivos aplicados a problemas reais, incluindo séries temporais para previsão de demanda de reservas, modelos de classificação, detecção de anomalias, análise de sentimentos e otimização de modelos com algoritmos genéticos. Também colaboro em testes e consumo de APIs de modelos preditivos e na integração desses resultados com sistemas de BI.
